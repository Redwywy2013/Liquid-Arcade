# Liquid Arcade

Liquid Arcade is a collection of browser-based games, simulations, puzzles, and tools built with HTML, CSS, and JavaScript. Each project uses a glowing glass-style interface with dark panels, soft neon highlights, rounded controls, and responsive layouts.

The goal of the project is to make browser applications feel polished, interactive, and visually satisfying while staying lightweight and fully frontend-based.

## Live Site

https://wyr3forrealz.org

# Projects

## Liquid Electrons

An interactive electronics sandbox for building and testing simple circuits.

Route: /liquids/electrons/

## Liquid Snake

A smooth neon version of Snake with custom settings, keybinds, scoring, and glowing arcade visuals.

Route: /liquids/snake/

## Liquid Universe

A gravity sandbox for spawning planets, stars, black holes, and chaotic orbital systems.

Route: /liquids/universe/

## Liquid Calc

A scientific calculator with variables, custom functions, memory, history, and advanced math tools.

Route: /liquids/calc/

## Liquid 2048

A glowing version of 2048 with sliding tiles, merging powers of two, and smooth puzzle gameplay.

Route: /liquids/2048/

## Liquid Minesweeper

A red-themed Minesweeper puzzle where hidden mines, clue numbers, flags, and logic decide whether the grid is cleared or detonated.

Route: /liquids/minesweeper/
