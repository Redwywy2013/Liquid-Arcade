# Liquid Arcade

Liquid Arcade is a collection of browser-based games, simulations, and tools built with HTML, CSS, and JavaScript. Each project uses a glowing glass-style interface with dark panels, soft neon highlights, rounded controls, and responsive layouts.

The goal of the project is to make browser applications feel polished, interactive, and visually satisfying while still remaining lightweight and fully frontend-based.

# Projects

## ? Liquid Electrons

Liquid Electrons is an interactive electronics and circuit sandbox designed to simulate simple electrical systems directly in the browser.

Players can place batteries, resistors, switches, LEDs, motors, buzzers, and additional components onto a workspace and connect them together to form functioning circuits.

The project focuses on experimentation and visual feedback, making it possible to test ideas and watch circuits behave dynamically without requiring physical hardware.

### Features

- Interactive component placement
- Wiring and circuit connections
- Live visual feedback
- Adjustable component behavior
- Glowing liquid-style UI
- Browser-based electronics experimentation

### Route

/liquids/electrons/

## ?? Liquid Snake

Liquid Snake is a modern neon-style recreation of the classic Snake arcade game.

The project expands the original gameplay with smoother movement, customizable visuals, editable controls, score tracking, adjustable speeds, and configurable game settings.

The focus is on making a familiar arcade game feel polished and satisfying while keeping the gameplay simple and responsive.

### Features

- Smooth arcade movement
- Adjustable board sizes
- Editable keybinds
- Speed customization
- Score tracking
- Neon liquid-style visuals

### Route

/liquids/snake/

## ?? Liquid Universe

Liquid Universe is a gravity and orbital physics sandbox.

The project allows users to create planets, stars, black holes, and other celestial objects while simulating gravity, collisions, orbital mechanics, and large-scale interactions.

Objects can orbit each other, collide, merge, or spiral into chaos depending on their mass, velocity, and position.

The simulation is designed to be both visually interesting and fun to experiment with.

### Features

- Gravity simulation
- Orbital mechanics
- Planet and star creation
- Black holes and collisions
- Physics experimentation
- Real-time simulation controls

### Route

/liquids/universe/

## ?? Liquid Calc

Liquid Calc is a glowing scientific calculator built for advanced calculations and mathematical experimentation.

The calculator includes scientific functions, variables, custom functions, memory storage, expression parsing, and additional tools for more advanced operations.

Unlike a simple calculator, Liquid Calc is designed more like a browser-based math workstation.

### Features

- Scientific calculations
- Variables and memory
- Custom functions
- Trigonometry support
- Expression parsing
- Advanced mathematical tools
- Responsive liquid-style interface

### Route

/liquids/calc/

## ?? Liquid 2048

Liquid 2048 is a neon-styled version of the classic 2048 puzzle game.

Players slide tiles across the board to merge matching values into larger powers of two while managing limited space.

The project keeps the original gameplay loop while adding smoother visuals, glowing effects, and a cleaner interface.

### Features

- Sliding tile gameplay
- Tile merging mechanics
- Increasing difficulty
- Score progression
- Smooth animations
- Neon liquid-style visuals

### Route

/liquids/2048/

# Development

Clone the repository:

git clone https://github.com/Redwywy2013/Liquid-Arcade.git

Enter the project directory:

cd Liquid-Arcade

Push updates:

git add .
git commit -m "update"
git push

Cloudflare Pages will automatically rebuild and redeploy the live site whenever updates are pushed to the main branch.
